## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/CodingListBigC/Factory-Analytics.git

cd Factory-Analytics

python -m venv .venv
. .venv/bin/activate

pip install -r requirements.txt
```

## Ussage

To run the main project run.

```bash
python main.py
```
